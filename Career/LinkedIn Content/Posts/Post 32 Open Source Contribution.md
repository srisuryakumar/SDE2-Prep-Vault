Made my first open source contribution this week.

Repository: TheAlgorithms/Java [link to PR]
Contribution: Added Fenwick Tree (Binary Indexed Tree) implementation
with documentation and unit tests.

It took 3 hours. The PR was reviewed and merged in 48 hours.

What this means on a profile:
→ GitHub shows a green square on the contribution graph from a public repo
→ The repository has 57K stars — any visitor to my profile sees the merge
→ The PR itself demonstrates: clean code, unit tests, documentation

What I learned from the review:
The maintainer asked me to add a time complexity comment above each method.
Obvious in retrospect. Added to every future piece of code I write.

For anyone prepping: find a good-first-issue on any popular Java repository.
The contribution matters less than the habit of shipping to external codebases.
