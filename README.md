# SDE-2 Interview Preparation Vault

This repository contains the Obsidian Vault used for tracking and structuring my SDE-2 interview preparation. It follows a multi-phase structure and heavily utilizes metadata to track progress via dashboards.

## Folder Structure Overview

- **Dashboard**: High-level views of progress, applications, and revision.
- **Daily Journal**: Daily logs and weekly reviews.
- **Knowledge**: Atomic concepts across different subjects (Java, DSA, LLD, System Design, etc.).
- **Practice**: LeetCode problems, mock interviews, and project hubs/ADRs.
- **Career**: Company dossiers, applications, interview debriefs, and networking.
- **Templates**: Standardized formats for different notes.
- **_Source-Archive**: Legacy files and notes prior to the Obsidian migration.

## Required Plugins

To fully utilize this vault, the following Obsidian community plugins are required:
1. **Dataview**: Powers every dashboard and MOC query in this system.
2. **Templater**: Enables dynamic fields (today's date, file title) inside templates.
3. **Tasks** (Optional but recommended): For tracking open action items.
4. **Obsidian Git** (Optional): Auto-commits and pushes to maintain a contribution record.
