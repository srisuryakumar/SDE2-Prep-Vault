# What is an Algorithm?

An **algorithm** is a finite, ordered sequence of unambiguous instructions that transforms an input into an output and is guaranteed to stop.

## Key Characteristics
- **Finite**: It terminates. "Keep checking" is not a step; "check up to 100 times" is.
- **Ordered**: Steps happen in a defined sequence, including branches and loops.
- **Unambiguous**: Each instruction means exactly one thing to whatever executes it.

"Find the largest number in this list" is a *problem*. The recipe that solves it is the *algorithm*. Every pattern in DSA (two pointers, sliding window, DP) is just a more sophisticated algorithm to solve a specific class of problems efficiently.
