# External Resources

### Books
- **[[A Mind for Numbers]]** by Barbara Oakley: Neuroscience of effective learning. Spaced repetition, sleep, focused vs diffuse thinking.
- **[[Effective Java]]** (3rd Edition) by Joshua Bloch: Java best practices.
- **[[Java Concurrency in Practice]]** by Brian Goetz: Thread safety, Java Memory Model, concurrency.
- **[[Cracking the Coding Interview]]** by Gayle McDowell: Interview process and mindset.
- **[[Elements of Programming Interviews in Java]]** by Aziz, Lee, Prakash: 250 problems with full Java solutions.
- **[[Designing Data-Intensive Applications]]** (DDIA) by Martin Kleppmann: Distributed data, replication, partitioning, consensus.
- **[[Database Internals]]** by Alex Petrov: Storage engines, B-Trees, LSM-Trees.
- **[[Clean Code]]** by Robert C. Martin: Writing readable, maintainable code.
- **[[Head First Design Patterns]]** by Freeman & Freeman: GoF patterns with exercises.
- **[[System Design Interview Vol 1]]** by Alex Xu: Basic system design framework and systems.
- **[[System Design Interview Vol 2]]** by Alex Xu: Advanced system design patterns and systems.
- **[[Spring in Action]]** (6th Edition) by Craig Walls: Practical Spring Boot reference.
- **[[Building Microservices]]** (2nd Edition) by Sam Newman: Modeling services, deployment, resilience.
- **[[The Pragmatic Programmer]]** by Hunt & Thomas: Software craftsmanship and career philosophy.
